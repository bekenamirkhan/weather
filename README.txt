Для приложения потребуется установить следующие библиотеки:

tkinter: это библиотека для создания графического пользовательского интерфейса (GUI).
Установка: pip install tkinter

requests: это библиотека для выполнения HTTP-запросов.
Установка: pip install requests

PIL (Python Imaging Library): это библиотека для работы с изображениями.
Установка: pip install pillow
